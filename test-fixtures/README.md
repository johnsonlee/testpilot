# Test Fixtures

This directory contains test APK files for integration testing.

## Download Test APK

The test APK is not committed to git due to size. Download it using:

```bash
./gradlew downloadTestFixtures
```

Or manually download from:
- Simple Calculator APK: https://f-droid.org/repo/com.simplemobiletools.calculator_50.apk

Place the downloaded file as `simple-app.apk` in this directory.
